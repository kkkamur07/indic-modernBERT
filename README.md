## Indic Modern BERT 

The goal of the project is to extend the modernBERT paper ( https://arxiv.org/abs/2412.13663 ) to 15 Indic Languages so that people can benefit from higher retrieval quality as a consequence of better sequenece length. 


The first approach is to train a good tokenizer for Hindi only, as then we can extend the same recipe to other indic langaugaes and combine the tokenizers if required.
